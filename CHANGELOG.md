# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2024-07-16

### Added
- Initial release of Vulhub-Web
- Zero-config Vulhub vulnerability library import with incremental scanning
- One-click lab lifecycle management (build, start, stop, destroy)
- Real-time container log streaming via SSE
- Interactive web terminal (xterm.js + WebSocket + PTY)
- Docker resource monitoring (disk, images, containers)
- Scheduled tasks: auto-scan, cache cleanup, idle reclamation
- JWT-based authentication with bcrypt password hashing
- Dark theme UI with responsive design
- 30+ vulnerability categories with auto-classification
- CVE extraction and year-based filtering
- Port conflict detection on lab startup
- Batch destroy for running labs
